# 化学基礎 50分レポート完結アプリ (Science Report App)

## 目的・ターゲット

- **目的**: 高校1年生が「化学基礎」の1年間の学びを振り返り、50分の授業内で自分なりの視点を持った質の高いレポートを完成させること。
- **ターゲット**: 高校1年生（白紙のレポート用紙を渡されると筆が止まってしまう生徒）
- **利用シーン**: 年度末のまとめの授業（50分）、各自の端末（Chromebook/iPad等）でブラウザからアクセスして利用。

## MVPの範囲（最小構成）

- **バックエンド無し・完全フロントエンド完結**: 生徒の思考を止めないよう、ログイン等の無駄なステップを排除。入力データはブラウザ上のみで完全に保持。
- **ステップ・バイ・ステップ入力UI**:
  1. **STEP1 (テーマ選択)**: 大項目から興味のあるテーマを選ぶ。
  2. **STEP2 (事実の整理)**: 授業で学んだ「科学的な事実」を箇条書きでまとめる。
  3. **STEP3 (生活への応用)**: その科学技術が、私たちの生活でどう役立っているか？または問題を起こしているか？
  4. **STEP4 (自分の考え)**: これからの未来、その科学とどう付き合っていくべきか？
- **ヒント提示機能**: 各ステップで「何を書けばいいか」の例文や問いかけをプレースホルダーとして配置。（レポートの質を高める最大の仕掛け）
- **PDF出力機能**: 全ステップ入力を終えると、自動でA4サイズのレポート形式にレイアウトされ、ブラウザ機能でPDFとして保存可能。

## 技術スタック

- **構造**: HTML5
- **スタイリング**: Vanilla CSS (生徒が集中できるモダンで洗練されたダーク/ライト対応の教育用UI)
- **ロジック**: Vanilla JavaScript (TailwindやReact等の外部フレームワークに依存せず軽量・高速に動作)
- **PDFエクスポート**: ブラウザの標準印刷機能 (`window.print()`) ＋ 印刷用CSS (`@media print`) による高度なレイアウト制御
